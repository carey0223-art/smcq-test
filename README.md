# smcq-test
screening test for dementia
# 🧠 SMCQ (주관적 기억감퇴 평가 설문) 자가진단 도구

## 📜 프로젝트 소개 및 목적

이 프로젝트는 **SMCQ (Subjective Memory Complaints Questionnaire)** 기반의 **주관적 기억감퇴(SMC) 자가진단 웹페이지** 구현 코드입니다.

노인층 및 일반 사용자가 스스로 기억력 저하 정도를 손쉽게 점검하고, 인지 건강 관리의 필요성을 조기에 인지할 수 있도록 돕는 스크리닝 도구로 활용하는 것을 목적으로 합니다.

## ⚠️ 중요 주의 사항 (면책 조항)

### 1. **본 검사는 의학적 진단이 아닙니다.**
이 도구는 공식적인 **의료 진단**을 대체할 수 없습니다. 결과는 오직 **참고 자료** 및 **선별 목적**으로만 사용해야 합니다.

### 2. **전문가 상담 필수**
검사 결과 **6점 이상 (고위험군)**이 나온 경우, 자가진단 결과에 의존하지 마시고 반드시 **가까운 치매안심센터** 또는 **신경과/정신건강의학과 전문의**와 상담하여 정밀 진단(신경인지검사, 뇌영상검사 등)을 받으셔야 합니다.

### 3. **검사 결과 해석 및 지원 정보**
* **주관적 기억감퇴 (SMC):** 6점 이상은 스스로 느끼는 기억력 저하가 높다는 의미이며, **경도인지장애(MCI)**로 진행될 위험이 정상인보다 높습니다.
* **치매안심센터 연계:** 치매안심센터는 선별 검사를 제공하며, 병원 연계 시에도 **SNSB 등의 비급여 정밀 검사 비용은 전액 지원되지 않을 수 있습니다.** 검사비 지원은 **기준 중위소득 120% 이하** 등 소득 기준과 지원 상한액이 적용되므로, 센터에 직접 문의하여 확인해야 합니다.
